## Overview
This project is a Machine Learning assignment completed using Python in Google Colab. The objective is to analyze user subscription behavior, predict subscription renewals, and estimate monthly spending using various machine learning techniques.

## Tasks Performed
- Data loading and exploration
- Feature identification
- Missing value analysis
- Statistical analysis
- Categorical data encoding
- Train-test split
- Decision Tree Classification
- K-Nearest Neighbors (KNN) Classification
- Model evaluation using accuracy and confusion matrix
- Linear Regression for monthly spending prediction
- Business insights and recommendations

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Models Implemented
1. Decision Tree Classifier
2. K-Nearest Neighbors (KNN)
3. Linear Regression

## Dataset
The dataset contains user demographics, subscription information, engagement metrics, spending behavior, and subscription renewal status.
